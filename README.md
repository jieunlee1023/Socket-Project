# 채팅 프로그램
[ 🗣 소켓(Socket)통신 - 채팅 프로그램 ]
- 멀티 스레드, 다중 접속 구현, 소켓 통신을 이용한 채팅 프로그램


## 📋 상세 내용

- 소켓통신을 이용한 채팅 프로그램 입니다. 
- 서버와 클라이언트간에 프로토콜을 주고 받으며 양방향, 다중 접속이 가능하게 구현하였습니다. 
- 같은 서버, 같은 방 유저끼리만 채팅이 가능하며 귓속말도 보낼 수 있습니다. 
- 화면은 네이트온과 비슷한 느낌으로 구현을 해보았습니다.
