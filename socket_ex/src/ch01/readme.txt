소켓 통신이란 ?
소켓이라는 것인 운영체제 이미 존재한다.
양 끝단에 포트를 달아서 byte stream (0,1의 데이터) 으로 통신을 주고 받는 것 

소켓 통신 종류
1. 1 : 1 통신 (일대일 소켓 통신, 단방향 소켓 통신)
서버 소켓 생성, 클라이언트 소켓 생성 (ip 주소와 포트를 사용해서 연결 할 수 있다.)

소켓 통신 중에 서버 소켓 (host ip, port) - 방화벽 허용이 되어있어야 한다.
클라이언트 소켓 (접속 할 host 주소, 포트 번호) - 네비게이션 주소, 상세 동 ..
연결은 스트림을 통해서 연결 한다.
